# AI 足迹 — 互联网足迹，AI 分类整理

**AI 足迹**是一款面向普通用户的智能收藏管理工具。不只是「存链接」——粘贴任意网页链接，AI 自动提取内容、分类、打标签，帮你把散落在各平台的收藏汇聚一处。

🌐 **官网：** [ai.ocean94.com](https://ai.ocean94.com)

---

## 三大亮点

### 🌐 跨平台 · 全收纳

小红书攻略、大众点评餐厅、淘宝装备、B站视频……不管来自哪个平台，一个链接丢进来，AI 自动归类。同一个主题（如「旅行」）的内容终于可以放在一起，不再散落各处。

### 👥 共享协作

创建共享分类，实时协作：
- **共创模式** — 成员都可以添加和编辑，适合团队知识库、旅伴做攻略
- **订阅模式** — 只读精选，适合分享你的精选清单给朋友

### 🤖 AI Agent 原生接入

不只是人类用的工具。生成访问令牌，你的 AI 助手就可以管理收藏——搜索、归类、整理、备份。支持双端访问：
- **PC 端**（推荐）— 可设为浏览器首页，全文搜索，暗色/亮色双模式
- **手机 H5** — 浏览器直接访问，无需下载 App
